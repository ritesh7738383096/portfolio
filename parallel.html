About This Project

This page demonstrate how to create a clean parallax scrolling effect using Bootstrap 3 and a bit of CSS. It's great for learning how to build visually interesting one-page websites.

The parallax effect happens when the backgroung image stays fixed while the foreground content scrolls-giving a nice sense of depth to your website design.
